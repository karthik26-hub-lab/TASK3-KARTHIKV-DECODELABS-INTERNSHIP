# Travique - Minimalist Concierge ✈️

Travique is a modern, minimalist travel concierge application. This project is currently transitioning from Phase 2 (Express.js Backend setup) into Phase 3 (Database Integration).

## 🚀 Features
* **Interactive Frontend:** Built with Vanilla JavaScript, HTML5, and CSS. Features a sleek onboarding flow, an "Explore" grid, and a "Journal" for saved destinations.
* **API Integration:** The frontend is preparing to communicate with a custom Node.js/Express.js backend.
* **Curator Form:** Users can submit new "Gems" via the UI.
* **Modular Backend:** Clean MVC-style routing for scalability.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, Vanilla JavaScript
* **Backend:** Node.js, Express.js, CORS
* **Architecture:** RESTful API design
